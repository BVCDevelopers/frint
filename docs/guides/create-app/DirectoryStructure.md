# Directory Structure

We try to follow a convention for our file structure in Apps. A regular App's tree would look like this:

```
├── app/
│   ├── index.js
├── components/
│   ├── Root.js
├── constants/
├── actions/
├── reducers/
├── services/
├── factories/
├── .babelrc
├── index.js
├── webpack.config.js
└── package.json
```

We will go through all of them next.
